## 4️⃣ Miniguia de Estudo

### Resumos Estruturados
**Conceito 1: Modelos de Linguagem**  
- Modelos treinados para gerar texto coerente.  
- Exemplos: GPT, BERT, LLaMA  

**Conceito 2: Difusão em IA**  
- Técnica para gerar imagens a partir de descrições.  
- Aplicações: DALL·E, Stable Diffusion  

### Glossário
- **IA Generativa:** Inteligência Artificial capaz de criar conteúdo novo a partir de dados.  
- **Prompt:** Instrução ou pergunta feita à IA para obter uma resposta.  
- **Fine-Tuning:** Ajuste fino de modelos de IA em dados específicos.  

### Prompts Reutilizáveis
- "Resuma os principais conceitos de [tema] em tópicos claros"
- "Liste exemplos práticos de [conceito] com aplicações reais"
- "Crie um glossário dos termos mais importantes de [tema]"
